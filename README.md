# environment
python 3.6+
```shell
# create virtual python environment 
$ python3 -m venv venv
$ .\venv\Scripts\activate
$ pip install -r requirements.txt
```

if you pip install any module, please update the requirements.txt via the command below
```shell
$ pip freeze > requirements.txt
```
